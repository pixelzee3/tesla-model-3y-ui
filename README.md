# Tesla Model 3/Y UI

## Information
* The UI is responsive, but an 8:5 screen aspect ratio is recommended.

## Project Setup
```sh
npm install
```

### Compile and Hot-Reload for Development
```sh
npm run dev
```

### Compile and Minify for Production
```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)
```sh
npm run lint
```
